# Frontend Mentor - E-commerce Product Page

This is a solution to the Frontend Mentor **E-commerce product page challenge**.

🔗 Challenge link:  
https://www.frontendmentor.io/challenges/ecommerce-product-page-UPsZ9MJp6

Frontend Mentor challenges help developers improve their coding skills by building realistic projects.

---

## 📌 Overview

The goal of this project is to build a responsive **e-commerce product page** based on the provided design.

Users should be able to:

- View the optimal layout depending on their device screen size
- See hover states for all interactive elements
- Open a lightbox gallery by clicking the product image
- Switch the large product image using thumbnails
- Add items to the cart
- View the cart and remove items from it

---

## 🛠 Built With

- Semantic **HTML5**
- **CSS3**
- **JavaScript**
- Responsive design techniques
